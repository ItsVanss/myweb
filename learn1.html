Fungsi-Fungsi Library Math di Python

Kumpulan fungsi matematika yang berguna dalam bahasa Python

math.ceil(x)

Membulatkan angka ke atas ke bilangan bulat terdekat.

math.ceil(3.7)  # Output: 4

math.floor(x)

Membulatkan angka ke bawah ke bilangan bulat terdekat.

math.floor(3.7)  # Output: 3

math.sqrt(x)

Menghitung akar kuadrat dari x.

math.sqrt(16)  # Output: 4.0

math.pow(x, y)

Menghitung hasil x pangkat y.

math.pow(2, 3)  # Output: 8.0

math.exp(x)

Mengembalikan nilai e^x (eksponen basis e).

math.exp(2)  # Output: 7.3890560989306495

math.log(x, base)

Menghitung logaritma dari x dengan basis tertentu.

math.log(100, 10)  # Output: 2.0

math.sin(x)

Menghitung sinus dari sudut x dalam radian.

math.sin(math.radians(30))  # Output: 0.49999999999999994

math.cos(x)

Menghitung kosinus dari sudut x dalam radian.

math.cos(math.radians(60))  # Output: 0.5

math.tan(x)

Menghitung tangen dari sudut x dalam radian.

math.tan(math.radians(45))  # Output: 1.0

math.degrees(x)

Konversi dari radian ke derajat.

math.degrees(math.pi / 2)  # Output: 90.0

math.factorial(x)

Menghitung faktorial dari x.

math.factorial(5)  # Output: 120

math.gcd(x, y)

Menghitung faktor persekutuan terbesar (FPB) antara x dan y.

math.gcd(36, 60)  # Output: 12

math.trunc(x)

Menghapus bagian desimal dari x dan hanya menyisakan bagian bilangan bulat.

math.trunc(3.7)  # Output: 3
